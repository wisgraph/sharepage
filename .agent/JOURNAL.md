# Galpi 개발 저널 (Development Journal)

이 폴더는 Galpi 프로젝트의 일일 개발 계획과 진행 사항을 기록하는 공간입니다. 에이전트와 사용자 사이의 작업 맥락을 유지하고, 나중에 변경 사항을 추적하기 위해 사용합니다.

## 📝 관리 규칙
1. 매 세션 종료 전 또는 주요 작업 완료 후 해당 날짜의 파일(`YYYY-MM-DD.md`)을 생성하거나 업데이트합니다.
2. 진행 중인 계획, 완료된 작업, 마주친 문제점 및 해결 방안을 기록합니다.
3. 이 파일은 인덱스 역할을 수행합니다.
