# 갈피 앱 설치 가이드

## 다운로드 및 설치

다운로드 링크를 누르시면 갈피 앱을 다운 받으실 수 있습니다.
**aarch64** 는 m시리즈 맥북
**x86** 은 인텔 맥북에서 사용하실 수 있습니다
![[Pasted image 20260226135153.png]]

다운로드 후에 실행하시면
Applications 에 드래그 하여 설치할 수 있습니다.
![[Pasted image 20260226135134.png]]
응용프로그램에서 해당 앱을 찾아 실행하면
![[Pasted image 20260226135730.png]]
앱스토어 앱이 아니라고 이런 다이얼로그 창이 뜹니다.
갈피 앱은 다른 앱의 정보를 조회하고 활용하기 때문에
앱스토어 정책상 등록이 불가능하지만,
애플의 정식 승인을 받은 앱입니다. 안전하게 사용할 수 있습니다.
확인을 눌러주세요
![[Pasted image 20260226135754.png]]

## 첫 실행 & 글로벌 단축키 설정

설치 후 실행하면 웰컴페이지가 나타납니다.

![[Pasted image 20260226151458.png]]
만약 웰컴 페이지가 나타나지 않는다면 상단 메뉴바에서 갈피 아이콘을 클릭하고 열기를 누르시면 됩니다.
`v1.2.1`에 누락된 최초구동시 웰컴페이지 자동실행 로직을 `v1.2.2`에서 복원하였습니다.

![[Pasted image 20260226210638.png]]

갈피앱은 독아이콘으로 사용할 수 있지만(**설정 > 인터페이스 > 독아이콘표시**),
전역 단축키를 설정하여 사용하기를 권장하고 있습니다.
갈피앱은 모든 조작을 키보드로 할 수 있게 해놓은 키보드 친화적인 앱입니다.
시작 단축키는 다른 앱의 단축키와 충돌하지 않게 하기 위해
잘 사용되지 않는 `cmd + option + G` 키를 기본값으로 갖고 있습니다.
저는 `option + space` 를 사용하고 있습니다.
단축키 버튼을 클릭하여 키를 입력하면 자동 캡쳐됩니다.
![[Pasted image 20260226152006.png]]

## 시스템 권한 설정

갈피 시작하기 버튼을 누르면
이런 권한을 허용해달라는 다이얼로그가 뜨는데,
![[Pasted image 20260226164633.png]]

![[Pasted image 20260226170401.png]]
![[Pasted image 20260226164737.png]]
![[Pasted image 20260226164752.png]]
갈피 앱이 등록되고 활성화되어있어야 합니다.

만약 다이얼로그가 자동으로 뜨지 않는다면
**설정(`cmd+.)` > 시스템&데이터 > 시스템 설정**에서 열기 버튼을 누르시면 등록 가능합니다.
맥 자체 권한이 없으면 아래 스크린샷 처럼 다른 앱의 정보를 조회하는
접근성API나 애플스크립트의 사용이 제한되어
캡쳐가 진행되지 않습니다.
![[Pasted image 20260226165101.png]]
![[Pasted image 20260226164913.png]]
권한이 모두 활성화되면 브라우져나 파인더 옵시디언을 캡쳐시
![[Pasted image 20260226202921.png]]
옵시디언 문서 제목과 url링크를 확인 할 수 있습니다. URL링크가 잘 보인다면 정상캡쳐가 가능합니다
`Enter`를 누르거나 캡쳐카드를 마우스로 클릭하면
캡쳐된 카드를 로컬에 저장 합니다.
캡쳐카드의 우측 상단에 `new` -> `existing` 으로 변한것을 확인 할 수 있습니다.
![[Pasted image 20260226203007.png]]
이제, 옵시디언 카드에 마우스를 올리고 클릭하면, 옵시디언이 열립니다.

![[갈피 옵시디언 실행.gif]]

축하합니다!
설치에 성공하셨습니다.
