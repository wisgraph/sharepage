# 바이브 코딩 의사결정 지도

서비스 아이디어를 플랫폼과 기술스택으로만 연결하지 않고, 고객 획득 경로, 가격 모델, 데이터 민감도, 외부 연동, 운영 난이도, 대략적인 월 비용까지 한 번에 보도록 확장한 버전입니다.

**흐름**: 문제 → 채널 → 플랫폼 → 스택 → 가격 → 보안 → 비용
**기준**: 웹 우선 · 관리형 우선 · 무료 시작 가능성 표시

---

## 확장 의사결정 흐름도

왼쪽에서 오른쪽으로 따라가며, 예외가 있을 때만 앱·커스텀 백엔드·고비용 구조로 이동하도록 읽으면 됩니다.

### 범례

- **웹 축**: 문서, 대시보드, 업무 자동화, SEO, 공유형 서비스
- **앱 축**: 카메라, 위치, 푸시, 습관 루프가 핵심인 경우
- **비즈니스 축**: 고객 획득 경로와 가격 모델을 먼저 붙임
- **운영 축**: 데이터 민감도와 월비용에 따라 관리형/고급 운영으로 분기

## 월 비용 러프 가이드

아래 금액은 초반 MVP 기준의 대략적 출발선입니다. 실제 비용은 트래픽, 저장공간, API 사용량, 팀 좌석 수에 따라 달라집니다.

| 서비스 유형      | 권장 구조                            | 대략 월비용                | 비고                                         |
| ---------------- | ------------------------------------ | -------------------------- | -------------------------------------------- |
| 랜딩/정적 사이트 | Vercel Hobby + 정적 배포             | $0~$20                     | 실험은 무료 시작 가능, 유료 전환 시 Pro 검토 |
| 웹 SaaS MVP      | Vercel + Supabase Free/Pro           | $0~$45                     | 둘 다 무료 시작 가능, 안정 운영 시 Pro 조합  |
| B2B 운영툴       | Next.js + Supabase Pro + 관리형 인증 | $25~$70+                   | 트래픽·스토리지·MAU 따라 증가                |
| AI 보조 웹툴     | 웹 SaaS + OpenAI API                 | $25~$100+ 또는 사용량 비례 | 모델·토큰·출력량에 따라 크게 달라짐          |
| 모바일 앱 MVP    | React Native + Supabase/Firebase     | $25~$100+                  | 백엔드 외에 앱 배포·업데이트 운영 부담 존재  |

핵심은 "무료로 시작 가능한가"와 "유료 전환 시 기본 고정비가 얼마로 올라가는가"를 함께 보는 것입니다.

## 주요 서비스 요금

### 호스팅 — Vercel

- Hobby는 무료 시작점
- Pro는 월 $20부터

### 백엔드 — Supabase

- Free는 $0, DB 500MB와 MAU 50,000 포함
- Pro는 월 $25부터, 8GB DB와 MAU 100,000 포함

### AI API — OpenAI

- 텍스트 API는 토큰 기반 종량제
- 예시 가격 중 하나는 입력 1M 토큰당 $0.20, 출력 1M 토큰당 $1.25

## 무료 시작이 유리한 경우

| 조건                              | 의미                                     |
| --------------------------------- | ---------------------------------------- |
| 로그인 없이도 핵심 가치 확인 가능 | 정적 배포나 간단한 웹으로 먼저 검증 가능 |
| 파일 저장·트래픽이 적음           | 무료 플랜으로 꽤 오래 버틸 수 있음       |
| AI 호출을 데모 수준으로 제한      | 초기 API 비용을 매우 낮게 유지 가능      |
| 지인 영업·수동 온보딩 중심        | 마케팅 도구 비용 없이도 검증 가능        |
