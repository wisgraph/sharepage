# 📚 MOC (Map of Content)

이 문서는 **Map of Content(MOC)**의 예시입니다. MOC는 여러 문서를 주제별로 묶어 구조화된 목차를 제공하여 탐색을 돕는 역할을 합니다.

## 🚀 시작하기 (Getting Started)
SharePage를 처음 접하신다면 이 가이드들을 먼저 확인하세요.

- [[welcome]] : 환영 메시지와 기본 기능 소개
- [[사용자 가이드]] : 상세 기능 및 사용법 안내
- [[todo]] : 앞으로의 할 일 목록

## 🤖 인공지능 (AI Topics)
최신 기술 트렌드와 인사이트를 담은 아티클입니다.

- [[AI 혁명과 기술]] : AI 기술의 발전과 영향력에 대한 고찰
- [[신공지능]] : 새로운 인공지능 패러다임

## 🎨 디자인 및 스타일 (Design System)
SharePage의 디자인 요소와 스타일 가이드를 확인해보세요.

- [[style]] : 타이포그래피, 색상, UI 컴포넌트 예시
